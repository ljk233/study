# Selection and Iteration

## For loop

Syntax
: `for (iterator, condition, iterate iterator)`

~~~java
for (int k = 0; k < x; k++)
{
    // do something
}
~~~

## If selection

~~~java
if (aVar < x)
{
    // if this, then do that
}
else if (aVar < y)
{
    // else if this, then do that
}
else
{
    // else do that
}
~~~

## While loop

Syntax
: `while (condition)`

~~~java
while (aVar > x)
{
    // do somethng
    // ensure the condition will
    // eventually become false
}
~~~
